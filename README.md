# Ferns and Petals Sales Performance Analysis

*Project Overview

This project provides an in-depth analysis of Ferns and Petals' sales data, focusing on key performance metrics and trends.

*Key Objectives:

1. Total Revenue: Comprehensive view of overall sales revenue.

2. Average Order and Delivery Time: Insights into order fulfillment efficiency.

3. Product and Category Performance: Identification of top-performing products and categories.

4. Geographical Insights: Analysis of sales distribution across cities.

5. Occasion-Based Performance: Evaluation of sales trends by occasion.

*Features:

1. Single Slicer for Occasion: A unified slicer that dynamically filters multiple pivot tables based on the selected occasion.

2. Interactive Dashboard: Seamless navigation and filtering for in-depth analysis.

3. Comprehensive Visualizations: Including bar charts, line charts, and KPI cards to highlight key metrics.

*How to Use:

1. Open the Dashboard: Load the Excel file to view the interactive dashboard.

2. Select an Occasion: Use the occasion slicer to filter data across multiple pivot tables.

3. Explore Insights: Analyze the visualizations to understand sales trends and performance metrics.
